# FUNKTION
Der RESPO-BEAM-POINT definiert den exakten Bindungspunkt,
an dem der RESPO-Interaktionator innerhalb der LAGE-Struktur wirkt.
Er steuert die Aktivierung, Neutralisierung und Weiterleitung
von RESPO innerhalb der Station LAGE und dem Ort ROOT / Creator / NC.
